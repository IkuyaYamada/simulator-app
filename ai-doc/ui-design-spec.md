# UI設計仕様書

## 全体構成

### アプリケーション概要
**株式投資シミュレーション・学習アプリケーション**
- 投資戦略の仮説検証と学習支援
- AIリサーチ連携による銘柄選定支援
- チェックポイント方式による投資判断記録
- 自動売買条件監視とポートフォリオ追跡

## レスポンシブ対応

### モバイル表示
- カード形式での情報表示
- スワイプ操作での画面遷移
- 重要な情報の優先表示
- タッチ操作に最適化されたボタンサイズ

### タブレット表示
- 2カラムレイアウトの活用
- チャートと詳細情報の並列表示
- より大きなタッチターゲット

## アクセシビリティ

- キーボードナビゲーション対応
- スクリーンリーダー対応
- カラーコントラストの確保
- フォントサイズの調整機能

## 技術スタック

- **フロントエンド**: React (Remix)
- **UI フレームワーク**: Tailwind CSS
- **チャート**: Chart.js / Recharts
- **アイコン**: Heroicons / Lucide React
- **状態管理**: React Query + Zustand

## 実装優先順位

### Phase 1: 基本機能
1. ダッシュボード
2. 銘柄検索・AIリサーチ
3. シミュレーション作成
4. シミュレーション詳細

### Phase 2: 高度な機能
1. チェックポイント作成
2. 自動監視・通知
3. レビュー・学習機能
4. データ可視化の強化

### Phase 3: 拡張機能
1. 複数銘柄ポートフォリオ
2. バックテスト機能
3. 学習データベース検索
4. ソーシャル機能（共有・比較）
